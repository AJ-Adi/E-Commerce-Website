Troubleshooting Tips

  • Make sure all images are in the DEMO/img/ folder
  • Check that the filenames match exactly: lowercase and correct extension (.jpg)
  • Ensure images are properly formatted JPG files (not text files with .jpg extension)
  • Try using 500×500px square images for best results
  • Image file size should ideally be under 300KB
